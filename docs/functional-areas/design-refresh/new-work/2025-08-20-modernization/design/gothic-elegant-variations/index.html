All designs include:

  • Organic Layout: Clip-path curves and flowing sections inspired by original Rope & Flow
  • Underline Animations:
    • Navigation menu items (center outward)
    • Primary CTA buttons (under button text)
    • Event tiles (between image and text content)
    • Footer links (left to right)
  • NO Mouse Following: All animations are static or triggered by hover/focus only
  • Accessibility: WCAG 2.1 compliant with proper focus indicators
  • Responsive: Mobile-first design with touch-friendly interactions
  • Performance: CSS-only animations with reduced motion support

Ready for Mantine v7 implementation: All designs use standard web components that can be easily converted to Mantine components with the same styling approach.
